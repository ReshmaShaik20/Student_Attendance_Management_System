## 📌 Project Overview
This is a Student Attendance Management System built using the MERN stack (MongoDB, Express, React, Node.js).  
It allows attendance tracking using facial recognition and student database management.

## 🚀 Features
- Student registration
- Attendance marking using webcam
- Face recognition-based attendance (Face API)
- Attendance history tracking
- Admin dashboard
- Student database management

## 🛠️ Tech Stack
- Frontend: React, JavaScript, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Other: Face-api.js, Axios, Webpack

## 📂 Project Structure
- frontend/ → React UI
- backend/ → Node + Express server
- database → MongoDB collections
